Our Technologies
WAW wramework
is an open source, fast and simple framework for creating mobile and web applications. Easy to use to create complex projects with the highest quality.
Vue JS
JavaScript framework that uses the MVVM template to create user interfaces based on data models through reactive data binding. Vue uses HTML-based template syntax to declaratively associate DOM rendering with basic data instances in Vue.
Angular
JavaScript framework written in open source typeScript developed by Google. The main elements in the development are modules, components, templates, metadata, directives and services.
React
is an open source, fast and simple JavaScript framework for creating user interfaces. React allows you to create web applications that use data that changes over time without reloading the page.
Node JS
is an open source JavaScript platform for high-performance tasks. Node.js provides the ability to execute JavaScript code on the server and send to user the result of their execution.
Express JS
is the software basis for developing the server part of web applications for Node.js in which open source software is implemented. It is designed for creating web applications and APIs.
Mongo db
is a document-oriented open source database management system. MongoDB handles data in a key / value format that is functional and easy to query. MongoDB supports automatic disaster recovery

Workflow
Once a new project is submitted, the party actually starts. Our developers, designers and universal agents cooperate together on one goal - to create a high level product. In order to follow the trends, they constantly discuss the ideas and share the suggestions for improvements
Designers
A designer is responsible for creative visual work. Everything that is visible to end customers, initially was made up by this worker. It can seem that the design consists of details like fonts, stickers, layouts, frames etc. Though, our designers also absorb all visual trends and deploy them in fresh and exciting ways
Developers
A developer is an individual who builds software and applications. Mainly, this member is responsible for producing clean, efficient code based on client’s needs. The key duties are website flow, database scheduling, bugs fixing and new features implementing
Agents
Before any project or startup begins, it is essentially important to test it, considering all risks. That is what agents are responsible for: analyzing the idea, needs and competitors, creating fake data as well as collecting all the feedbacks after release to improve the product. Their role is shadowed but powerful as they always face the first line and can influence the quality of the project.
